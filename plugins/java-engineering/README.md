# java-engineering

Skills, agents, and hooks for Java backend work (Quarkus, Spring Boot, Timefold).
See the root README for install/dev instructions.

Usage notes:
- Skills auto-invoke; force one with /java-engineering:quarkus-conventions etc.
- Ask "review my changes" to trigger the code-reviewer agent; "write tests for X"
  for test-writer.
- The Lombok/MapStruct hook blocks at write time — Claude will self-correct when
  it sees the block message.
